<h1>PROJETO BABEL: Ultrapassando Barreiras</h1> 
<p>Este é o desenvolvimento de uma aplicação Web em que explica o funcionamento e aconstrução do meu TCC (Trabalho de Conclusão de Curso), onde desenvolvi um dispositivo de tradução de idiomas a partir de um script em Python e um SBC (Single Board Computer) Raspberry Pi 4B.
</p>  

<h2>Como funciona?</h2>
  <p>
    O site dispõe de um sumário onde consta todos os conteúdos abordados durante a produção do projeto, onde cada etapa mostra as dificuldades desenvolvidas assim como as suas soluções como:
    <ul>
      <li>Escolha da idéia</li>
      <li>Consolidação da Metodologia</li>
      <li>Referêncial Teórico</li>
      <li>Desenvolvimento do Script</li>
      <li>Dificuldades encontradas</li>
      <li>Resultados</li>
      <li>Conclusão</li>
    </ul>
  </p>

  <h2>Objetivos do Projeto</h2>
  <p>Desenvolver esta aplicação fará com que eu aprofunde meus conhecimentos em práticas Web, auxiliando na lógica, sintaxe e introdução de cada novo elemento incluído, tornando esta uma experiência .</p>
